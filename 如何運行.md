# 運行指南

## ❌ 為什麼 Live Server 不能用？

這個專案使用 **React + Vite**，需要：
1. 編譯 TypeScript → JavaScript
2. 處理 React JSX 語法
3. 打包模組依賴
4. 處理 Tailwind CSS

Live Server 只能開啟靜態 HTML，無法處理這些編譯工作。

## ✅ 正確的運行方式

### 步驟 1：打開終端機
在專案資料夾中打開終端機（Terminal）

### 步驟 2：啟動開發伺服器
```bash
npm run dev
```

### 步驟 3：打開瀏覽器
看到以下訊息後：
```
VITE v7.3.1  ready in XXX ms
➜  Local:   http://localhost:5173/
```

在瀏覽器打開：http://localhost:5173

## 🎯 快速指令

```bash
# 第一次使用（如果還沒安裝依賴）
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 💡 注意事項

- 開發伺服器運行時，終端機視窗不要關閉
- 修改代碼會自動熱更新（不需重新啟動）
- 按 `Ctrl + C` 可以停止伺服器
- 預設端口是 5173，如果被占用會自動換到 5174
