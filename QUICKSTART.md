# 快速啟動指南

## 台灣鞋品協會展示網站

### 🚀 快速開始

1. **安裝依賴**
```bash
npm install
```

2. **啟動開發伺服器**
```bash
npm run dev
```

3. **開啟瀏覽器**
訪問 http://localhost:5173

### 📱 功能說明

#### 首頁 (/)
- Hero 橫幅區塊
- 四大特色介紹（品質保證、安全購物、快速配送、專業服務）
- 精選商品展示（8款熱門商品）
- 合作品牌列表（8個品牌）
- CTA 行動呼籲區

#### 商品頁 (/products/:id)
- 商品圖片展示（可切換圖片）
- 商品詳細資訊
- 顏色選擇器
- 尺寸選擇器
- 購買按鈕（點擊顯示「此為靜態展示頁面」提示）
- 相關商品推薦

#### 品牌館 (/brands/:id)
- 品牌 Logo 展示
- 品牌介紹與歷史
- 成立年份與專業領域
- 聯絡資訊（Email、電話、網站）
- 品牌所有商品列表

### 🎨 設計特色

- **配色方案**：灰白藍三色搭配
- **字體**：系統預設字體，確保中文顯示良好
- **圓角**：適度的圓角設計（lg/xl）
- **陰影**：細膩的陰影效果（sm/md）
- **動畫**：流暢的過渡效果（300ms）
- **響應式**：完整支援手機、平板、桌面

### 📊 數據內容

- **8 個品牌**：涵蓋運動、商務、休閒、兒童等不同類型
- **20 個商品**：包含完整資訊（圖片、價格、描述、顏色、尺寸）

### 🔧 技術棧

- React 18
- TypeScript
- Vite 7
- Tailwind CSS 4
- React Router 7
- Lucide React (圖示)

### 📦 建置生產版本

```bash
npm run build
```

建置後的檔案將位於 `dist` 目錄，可直接部署到靜態網站託管服務。

### 🌐 推薦部署平台

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

---

**注意**：此為靜態展示網站，購買功能僅供展示用途，不具備實際交易功能。
